# TikTok Song Request Website 🎙️🎸

A sleek, real-time song request dashboard for TikTok creators. Features a fan-facing request portal and a unified "Control Center" for the creator.

## Features
- **Real-time Sync**: Songs appear instantly in the admin dashboard.
- **Control Center**: Manage the queue, "Now Playing", and "Up Next" labels.
- **Played Today History**: Fans can see what has been played in the last 24 hours and request encores.
- **Settings**: Toggle song requests on/off and update your live session name.

## Tech Stack
- Frontend: HTML, CSS, JavaScript (Vanilla)
- Backend: Supabase (PostgreSQL + Realtime)

## Setup
1. Create a Supabase project and a `requests` table.
2. Update the credentials in `app.js`.
3. Open `index.html` to start your stream!
